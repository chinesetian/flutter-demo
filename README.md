# myflutter

A new flutter-demo project.

## windows 环境配置

- 1，安装 flutter 环境依赖
- 2，安装 jdk、jre 依赖
- 3，安装 Android studio（提供安卓模拟器）
- 4，安装 vs code（运行flutter）
